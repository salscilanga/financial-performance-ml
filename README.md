# Predicting High-Performing Firms Using Machine Learning
This capstone project applies machine learning techniques to classify high-performing firms using financial ratios and company characteristics.

## Project Includes
- Exploratory Data Analysis (EDA)
- Statistical hypothesis testing
- Feature engineering
- Logistic Regression with PCA
- Random Forest
- Support Vector Machine (SVM)
- Tableau visualizations

## Key Findings
- Firm size was the strongest predictor of high performance
- Cash ratio showed a negative relationship with performance
- Logistic Regression + PCA achieved the highest recall (~93%)

## Tools Used
Python, Pandas, Scikit-learn, Seaborn, Matplotlib, Tableau.
